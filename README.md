# tasks-cli-rust
Project name: Task Management CLI

Components: 

+ Tasks
+ Lists
+ Tag
Feature: 


+ Add task.
	+ Each task is inside a list, and optionally have tags
+ View task: Has some basic metadata of each task, like:
	+ name
	+ description
	+ due date(optional)
	+ time spend in each task
+ Update task
+ Delete task
+ Export/Import
